# 📊 วิเคราะห์การปิดหนี้สินเชื่อบ้าน (Interactive)

เครื่องมือวิเคราะห์แผนปิดหนี้สินเชื่อบ้านแบบ Interactive ทำงานบน Browser 100% ไม่ต้องติดตั้งอะไรเพิ่ม

## ✨ ฟีเจอร์

- **แก้ไขข้อมูลได้ทันที** — เปลี่ยนยอดหนี้, ดอกเบี้ย, เงินโปะ แล้วกด "คำนวณใหม่"
- **แสดง Timeline** — เห็นภาพรวม Phase การปิดหนี้แบบ step-by-step
- **กราฟ Interactive** — ยอดหนี้ลดลงตามเวลา (Chart.js)
- **สรุปผลครบถ้วน** — ระยะเวลาปิดหนี้, ดอกเบี้ยรวม, ยอดผ่อนต่อเดือน
- **ไม่มีข้อมูลส่งออกไปที่ใด** — คำนวณทั้งหมดใน Browser ของคุณ

## 🚀 วิธีใช้งาน

1. เปิดไฟล์ `loan_analysis_interactive.html` ใน Browser
2. กรอกข้อมูลสินเชื่อ:
   - ยอดเงินต้นคงเหลือ
   - อัตราดอกเบี้ย (% ต่อปี)
   - ยอดผ่อนขั้นต่ำ/เดือน
   - เงินโปะเพิ่มต่อเดือน
3. กดปุ่ม **🔄 คำนวณใหม่**
4. ดูผลลัพธ์: KPI, สรุปแผน, กราฟ, Timeline

## 📁 ไฟล์

| ไฟล์ | รายละเอียด |
|------|-----------|
| `loan_analysis_interactive.html` | ไฟล์หลัก — เปิดใน Browser ได้เลย |

## 🛠️ เทคโนโลยี

- HTML / CSS / JavaScript (Vanilla)
- [Chart.js](https://www.chartjs.org/) — วาดกราฟ (โหลดจาก CDN)

## ⚠️ ข้อควรทราบ

- ข้อมูลนี้เพื่อ**ประกอบการตัดสินใจ** ควรปรึกษาที่ปรึกษาการเงินก่อนดำเนินการ
- ตรวจสอบเงื่อนไขการปิดสินเชื่อก่อนกำหนดกับธนาคาร (Prepayment Fee)
- ตรวจสอบสิทธิ์ดอกเบี้ยพนักงานหากมีการเปลี่ยนงาน
